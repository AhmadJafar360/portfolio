  • Software: Visual Studio Code
  • Framework: MERN Stack
  • Project URL: Flit Airlines

Penjelasan:

Final project adalah progress untuk mengasah kemampuan pengembangan, dan mengaplikasikan pengetahuan yang telah dipelajari secara keseluruhan mulai dari chapter 0 sampai chapter 8.

Final Project yang di kerjakan yaitu aplikasi bertema E-Flight Ticket Platform dengan nama aplikasi "Flit Airlines". Platform ini merupakan tempat beli dan booking tiket penerbangan secara online, baik one way (sekali penerbangan) dan round trip (pulang pergi). Platform ini membuka dan menyediakan berbagai jenis penerbangan domestik dan mancanegara. Flow / alur platform dibebaskan berdasarkan kreasi tim final project.

Tujuan terpenting dari proyek ini adalah menikmati proses pembuatan dan pembelajaran yang nantinya akan didapatkan bersama kelompok selama final project ini berjalan.
