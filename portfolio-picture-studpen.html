Short Story

Halo semuanya, Saya Ahmad Jafar Ali ini dari course Fullstack Web kelas 1.

Pada kesempatan ini, saya akan membagikan pengalaman yang saya dapatkan ketika mengikuti Studi Independen Di Binar Academy, sebelum saya menceritakan pengalaman saya, terlebih dahulu saya ceritakan alasan kenapa memilih Course Fullstack Web di Binar Academy.

Pada awalnya, saya hanya mahasiswa yang ingin mempelajari dunia IT karena teknologi sekarang semakin lama semakin modern & canggih. saat masuk ke dunia IT, banyak pemrograman yang saya tidak paham sama sekali karena ada banyak sekali bahasa pemrograman, saya tidak tau yang harus saya pelajari pemrograman yang mana. Saat masuk semester 2, saya mulai mengenal pemrograman web yaitu HTML CSS dan saya mulai tertarik untuk terjun ke bahasa pemrograman web. Awalnya saya membuat website secara manual yaitu html dan css, saat itu belum mengenal tentang framework dalam pembuatan website, pernah mengenal pemrograman PHP yaitu Laravel, bukannya paham tapi malah makin bingung..hehehe, karena saya saat itu masih pemula. Akhirnya saya stuck dan tidak tahu harus bagaimana, ingin menguasai bahasa pemrograman web tapi tidak tahu harus mulai dari mana, saya merasa salah masuk jurusan. Tetapi berkat dukungan dari orang tua, saya tidak mengeluh dan terus jalani apa yang ada di hadapan saya sekarang.

Sebelum ikut bootcamp, saya tiap hari-harinya ngoding HTML CSS secara manual, tapi makin lama semakin bosan karena ngoding cuma ini itu aja. Akhirnya saya bilang ke diri sendiri "Kalau kamu terus seperti ini, kamu tidak akan bisa mengembangkan skill, kamu harus mempelajari lebih dalam tentang pemrograman web", karena saya ingin membuat website bukan cuma tampilan (Frontend) saja, tapi juga bisa membuat dari sisi (Backend) yang dibutuhkan oleh frontend. Setelah itu, saya mendapatkan informasi kalau semester 7 yang harusnya kkn bisa di konversi ikut Studi Independen Kampus Merdeka, kemudian saya berinisiatif untuk ikut dan daftar di beberapa mitra yang menyediakan tentang pemrograman web. 2-3 hari kemudian, saya lolos pertama seleksi Studi Independen adalah Binar Academy (Fullstack Web) kemudian di susul mitra lainnya, awalnya ingin mengambil course fullstack walaupun agak ragu, tapi keraguan saya semakin kuat dan semakin bingung karena banyak omongan yang bilang "kalau mengambil fullstack kamu tidak akan bisa kuat, temenku yang ambil fullstack aja mengeluh, katanya susah banget karena bla bla bla bla hahahihi." Ya itu mereka bukan saya, untuk menghilangkan keraguan itu, saya tetap teguh pendirian dan menuruti keinginan untuk bisa pemrograman web dari frontend dan backend, karena yang penting adalah punya banyak pengalaman.

Nah, akhirnya saya memilih Binar Academy untuk Belajar, disini saya mempunyai pengalaman banyak dari Hardskill yang belajar dari Html5, Css Framework sampai ke framework ReactJs dan belajar NodeJs sampai ke framework ExpressJs dengan Fasilitator dari Industri sehingga Tech Communication & Presentation Skill, Analytical & Strategic Thinking juga mudah dipahami. Dan juga AAO yang membantu dan memberikan solusi saat student ada kendala selama bootcamp. Setiap hari Senin, Rabu, Jumat selalu ada Forum Discussion Materi dan tugas di setiap materi, kadang ada game dan quiz kecil yang berhadiah.

Terakhir saya mau menyampaikan terima kasih kepada "Kampus Merdeka" dan "Binar Academy" sebagai penyelenggara. Mas "Ahmad Hafidh Ayatullah" sebagai fasilitator yang telah memberikan saya banyak pengalaman dan ilmunya, Kak "Silvia S Saragih" sebagai AAO kelas yang telah membantu dan memberikan solusi saat ada kendala yang saya alami selama bootcamp. Teman-teman kelas Fullstack Web 1 yang telah membantu saat kesusahan pengerjaan challenge. Dan tak lupa saya berterima kasih kepada tim final Project @Firman Perdana, @Citra Meida Antika, @Novi Amanda sebagai frontend, @Dewa Biara, @Hendri Permana sebagai backend, dan @Irvan Wijaya @Zain Zaidan @Ro isul sebagai Android.
